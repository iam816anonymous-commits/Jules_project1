# Known Limitations: Jarvis OS v1 Beta

1. **Hardware Dependencies**: Real-world Voice (ASR/TTS) requires local mic/speaker configuration not fully available in all headless environments.
2. **Vision Latency**: High-resolution screen capture and OCR processing may add cycle latency on lower-end systems.
3. **Multi-Agent Orchestration**: Interaction between multiple specialized sub-agents is still in experimental stages.
4. **App Integration**: Support for specific 3rd party application GUI elements depends on ongoing Accessibility Tree mapping.
