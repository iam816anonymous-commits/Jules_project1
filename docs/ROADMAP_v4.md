# Roadmap: Jarvis OS Transition to Phase 4

## Implementation Order
1. **Foundation**: Documentation, Project Structure, Database Models.
2. **Core**: Runtime Kernel, Graph Engine, Policy Engine.
3. **Memory**: Working, Episodic, Semantic, and Learning layers.
4. **Planning**: Graph-based Planner and synthesis.
5. **Control**: Desktop, Window, and Browser controllers.
6. **Perception**: Vision (Screen capture/compression) and Voice (STT/TTS).
7. **Interface**: FastAPI Backend and React/Tauri Frontend.
8. **Hardening**: Observability, Chaos Testing, and Refinement.

## Milestones
- **M1**: Basic loop execution with state persistence.
- **M2**: Autonomous desktop navigation.
- **M3**: Voice-enabled multimodal interaction.
- **M4**: Self-healing and continuous learning.
